WATE is a package designed to work with the NPSS system to predict aircraft
propulsion system sizes and weights.  The WATE package is designed to work with'
an NPSS thermodynamic model of a propulsion system.  This version of the WATE 
package works with interprettted, .int, NPSS files.  This package of code is an 
add on.  It needs the general NPSS release to be installed on your computer to 
work.
